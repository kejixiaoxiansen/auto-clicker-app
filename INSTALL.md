# 自动点击器App - 安装指南

## 📦 快速安装

### 方法一：直接下载APK（推荐）

1. **下载APK文件**
   - 从 `output/AutoClicker-v1.0.0.apk` 获取安装包
   - 或通过以下链接下载：[下载链接]

2. **安装到手机**
   - 将APK文件传输到Android手机
   - 在文件管理器中点击APK文件
   - 允许"安装未知来源应用"
   - 完成安装

3. **开启必要权限**
   - 打开系统设置 → 无障碍 → 自动点击器
   - 开启无障碍服务
   - 允许悬浮窗权限（应用会提示）

### 方法二：从源码构建

#### 环境要求
- Node.js 18+
- Java JDK 11或17
- Android SDK (API 33+)
- Android Studio (可选但推荐)

#### 构建步骤

**Windows (CMD):**
```batch
build-apk.bat
```

**Windows (PowerShell):**
```powershell
.\build-apk.ps1
```

**手动构建:**
```bash
# 1. 安装依赖
npm install

# 2. 构建Android Release版本
cd android
./gradlew assembleRelease

# 3. APK输出位置
# android/app/build/outputs/apk/release/app-release.apk
```

## 🔧 系统要求

| 项目 | 要求 |
|------|------|
| Android版本 | 7.0 (API 24) 或更高 |
| 存储空间 | 50MB+ |
| 权限 | 无障碍服务、悬浮窗 |

## 📱 首次使用设置

### 1. 开启无障碍服务
```
设置 → 无障碍 → 已安装的服务 → 自动点击器 → 开启
```

### 2. 允许悬浮窗权限
应用首次启动时会自动请求，或手动设置：
```
设置 → 应用 → 自动点击器 → 显示在其他应用上层 → 允许
```

### 3. 关闭电池优化（可选）
确保应用在后台正常运行：
```
设置 → 电池 → 电池优化 → 自动点击器 → 不优化
```

## 🎯 功能使用

### 创建点击脚本
1. 点击首页右下角 **+** 按钮
2. 选择录制方式：
   - **录制点击** - 实时录制你的点击操作
   - **手动设置** - 按顺序点击设置位置
3. 点击保存按钮

### 播放脚本
1. 在首页点击脚本卡片
2. 点击播放按钮
3. 调整速度和循环次数
4. 点击开始执行

### 定时任务
1. 进入"定时任务"页面
2. 点击添加任务
3. 选择脚本和时间
4. 保存任务

## ⚠️ 注意事项

1. **安全警告**
   - 部分银行/支付类App会阻止自动点击
   - 请勿在敏感操作中使用（如转账、支付）
   - 使用风险自负

2. **稳定性提示**
   - 保持应用在后台运行
   - 关闭省电模式
   - 避免同时运行多个自动点击应用

3. **兼容性**
   - 部分游戏可能检测并阻止自动点击
   - 某些系统UI可能无法点击

## 🐛 故障排除

### 无法点击
- 检查无障碍服务是否开启
- 重启应用
- 重启手机

### 点击位置不准确
- 重新录制脚本
- 检查屏幕分辨率是否变化
- 关闭屏幕旋转

### 应用闪退
- 清除应用数据
- 重新安装
- 检查Android版本兼容性
