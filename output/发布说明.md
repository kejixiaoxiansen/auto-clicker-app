## 🚀 三种使用方式

### 方式一：直接安装APK（需要预编译APK）

如果你获得了预编译的APK文件：

1. 将APK传输到Android手机
2. 在手机上点击安装
3. 开启无障碍服务权限
4. 开始使用

**注意**: 本项目为源码发布，需要自行构建或使用CI服务构建APK。

---

### 方式二：本地构建APK

#### 环境要求
- Windows 10/11
- Node.js 18+ 
- Java JDK 11或17
- Android SDK (API 33+)

#### 快速构建步骤

1. **解压发布包**
   ```
   解压 AutoClicker-Complete-v1.0.0.zip
   cd auto-clicker-app
   ```

2. **运行构建脚本**
   ```powershell
   # PowerShell
   .\build-apk.ps1
   ```
   或
   ```batch
   :: CMD
   build-apk.bat
   ```

3. **获取APK**
   构建完成后，APK位于：
   ```
   output/AutoClicker-v1.0.0.apk
   ```

---

### 方式三：在线预览（无需构建）

直接在浏览器中打开 `demo/index.html` 即可预览App界面和交互效果。

## ✨ 功能特性

### 核心功能
- ✅ **录制点击** - 实时录制屏幕点击操作
- ✅ **手动设置** - 按1,2,3,4顺序点击确定位置
- ✅ **多脚本管理** - 保存、编辑、删除、复制多个脚本
- ✅ **调速播放** - 0.5x - 5x 速度调节
- ✅ **循环播放** - 支持设置循环次数或无限循环
- ✅ **定时任务** - 精确到毫秒的定时执行
- ✅ **定时重复** - 支持单次/每天/每周重复

### 技术特性
- React Native 0.72+ 跨平台框架
- TypeScript 类型安全
- Zustand 状态管理
- Android无障碍服务集成
- 悬浮窗权限支持
- 毫秒级定时精度

## ⚠️ 重要提示

### 安全警告
- 部分银行/支付类App会阻止自动点击
- 请勿在敏感操作中使用（如转账、支付）
- 使用风险自负

### 权限说明
本应用需要以下权限才能正常工作：
1. **无障碍服务** - 用于模拟点击操作
2. **悬浮窗** - 用于显示录制和播放控制界面
3. **后台运行** - 用于定时任务执行

### 兼容性
- 部分游戏可能检测并阻止自动点击
- 某些系统UI可能无法点击
- 不同手机厂商可能有额外限制

---

## 🐛 已知问题

1. **首次启动** - 需要手动开启无障碍服务
2. **后台限制** - 部分国产系统可能杀后台，需关闭电池优化
3. **分辨率变化** - 更换设备或旋转屏幕后可能需要重新录制
